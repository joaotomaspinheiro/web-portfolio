## Getting Started

```bash
# Clone repository
git clone https://github.com/joaotomaspinheiro/web-portfolio
cd web-portfolio
# Install dependencies
npm install
# Run server
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
